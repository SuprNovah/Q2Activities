Linked Styles

External style sheets or linked stylesare made outside the HTML file. An external style sheet islinked via <link>. The <link> element allows you yo establish document relationships.

The <link> tells the browser to find the specified style sheet. It can only be used to find the secifid style sheet. It can only be used within the ⁢head></head> of the document. The style sheet file does not contain any HTML code; it contains only style rules.
